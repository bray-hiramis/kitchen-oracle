# Kitchen Oracle

An online recipe finder.

___

## Features

Kitchen Oracle utilizes the *[themealdb](https://www.themealdb.com/api.php)* Free API.

- You can search meals
   - You can see the name of the meal,
   - Ingredients, and
   - Steps to cook
- You can also add a recipe to your favorite
- It uses persistent data so your favorites will be saved in your Browser Database (*sessionStorage*)

## Getting Started

> Note: The website is still under development and have few bugs.

1. Clone the repo to your preferred folder.

```bash
git clone https://github.com/bray-hiramis/kitchen-oracle.git
```

2. Open the folder (kitche-oracle) with your preferred editor (e.g. VSCODE).
3. Open the *index.html*
   - If you are in VSCODE, right-click on the code editor and click on Live Server.
   - You can also open it from the folder: 
      - right-click on the index.html, 
      - click on "Open With", 
      - and select the browser you want to open it (e.g. Chrome).
